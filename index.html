Code Insights

Code Insights is a data analytics project that explores the evolution of programming languages from 2000 to 2025. The project focuses on identifying which programming languages were most widely used across different years. Using the R programming language, I performed data cleaning, analysis, and visualization to uncover meaningful trends. Interactive bar charts and line graphs were created to visually represent language popularity over time. This project provides clear insights into how programming trends have evolved, highlighting dominant technologies and shifts in developer preferences. It demonstrates my ability to work with real-world data, perform analytical thinking, and present insights through effective visualizations.

NoteNest

NoteNest is a smart and user-friendly note-taking application designed to help users organize their thoughts, ideas, and important information efficiently. The platform allows users to create and manage notes that can include text, images, and audio recordings, making it more flexible than traditional note apps. Each note is automatically saved with a timestamp, helping users track when the content was created or updated. Notes are neatly organized for easy access and retrieval, improving productivity and information management. This project demonstrates my ability to design structured applications with multimedia handling, time-based organization, and user-focused functionality.

Chief’s Palette

Chief’s Palette is a web application built using React that helps users discover recipes based on the ingredients they have. Users can simply enter one or more ingredient names, and the application dynamically displays a list of recipes that can be prepared using those ingredients. This feature helps reduce food waste and makes cooking easier and more efficient. The application focuses on user-friendly design, fast search results, and practical meal planning, making it a helpful tool for everyday cooking.
